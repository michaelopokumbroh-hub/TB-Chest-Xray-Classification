# Mathematical Background
To evaluate and optimize the classification of Tuberculosis from Chest X-rays, the project utilizes the following mathematical frameworks.

1. Cross-Entropy Loss
Since this is a binary classification task, we utilize Cross-Entropy Loss. This function measures the performance of a classification model whose output is a probability value between 0 and 1.

2. Evaluation Metrics
Beyond simple accuracy, we calculate the F1-Score, which is the harmonic mean of Precision and Recall. This is the most reliable metric for medical datasets where class imbalance may exist.
